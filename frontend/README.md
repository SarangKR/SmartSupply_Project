# SmartSupply Frontend

This is the React + Vite frontend application for **SmartSupply**. 

It provides an interactive dashboard to visualize inventory levels, system alerts, and AI-driven demand forecasts generated by the backend.

## Quick Start

Make sure you have Node.js installed, then run:

```bash
npm install
npm run dev
```

The web application will be available at `http://localhost:5173`.

> **Note:** For the dashboard to fetch real forecast data, the SmartSupply Python backend must be running on port 8000. 

For full instructions on running the entire project stack (including the backend and machine learning pipeline), please see the [main project README](../README.md) in the root directory.

## Deployment

This frontend is configured for easy deployment on platforms like Vercel. 
If deploying to Vercel, make sure the **Root Directory** is set to `frontend` and specify your deployed backend URL in the `VITE_API_BASE_URL` environment variable.
